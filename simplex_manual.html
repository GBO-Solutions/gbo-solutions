Chapter 2: Theoretical Background

2.1 Standard Form of a Linear Programming Problem

A linear programming (LP) problem aims to optimize (maximize or minimize) a linear objective function, subject to a set of linear inequality or equality constraints. The standard form of a maximization LP problem can be written as:

Maximize z = c₁x₁ + c₂x₂ + ... + cₙxₙ
subject to:
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
...
aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ ≤ bₘ
x₁, x₂, ..., xₙ ≥ 0

This form assumes that all variables are non-negative, and all constraints are of the "less than or equal to" type.

2.2 Feasible Region and Vertices

The set of all possible solutions that satisfy the constraints is called the feasible region. Geometrically, it is a convex polyhedron formed by the intersection of the constraint inequalities. Each vertex (or corner point) of this region corresponds to a basic feasible solution.

2.3 Optimality Principle

The Simplex method works by moving from one vertex to another along the edges of the feasible region, always improving (or at least not worsening) the objective function. Due to the convexity of the feasible region and the linearity of the objective function, if a maximum exists, it will occur at one of these vertices.

2.4 Simplex Tableau

The Simplex algorithm is typically implemented using a tabular format known as the Simplex tableau. Each iteration updates this tableau by performing pivot operations, selecting an entering and a leaving variable, and computing a new basic solution.

2.5 The Algorithm

The Simplex method proceeds iteratively using the following general steps:

  1. Convert the problem to standard form (add slack variables).
  2. Construct the initial Simplex tableau.
  3. Select the entering and exiting variables based on improvement criteria.
  4. Perform the pivot operation and update the tableau.

The process repeats until termination is reached—when no further improvements can be made and optimality is achieved.

Chapter 1: Introduction

Linear Programming (LP) is a mathematical method used to determine the optimal outcome in a model defined by linear relationships. The Simplex method is a widely used algorithm for solving such problems, especially when the number of variables or constraints is large. It finds application in areas such as production planning, logistics, transportation, and financial optimization.

The purpose of this manual is to provide a concise but complete introduction to the Simplex method. It presents the theoretical foundations, a fully worked numerical example, and a functioning Python implementation. The reader will have the opportunity to understand the method and obtain a complete tool for applying it to real-world problems.
